✅. ANOVA (Analysis of Variance)
언제 사용하나요?
세 그룹 이상(A/B/C 등)을 비교할 때.

종류

One-way ANOVA: 한 요인 기준으로 그룹 간 차이 분석

Two-way ANOVA: 두 요인을 동시에 고려

예시
세 가지 디자인(버전 A, B, C) 중 클릭률 차이 있는지 확인

✅ 2. Bayesian A/B Testing
전통적인 빈도주의 통계 대신 베이즈 정리를 활용

장점

결과 해석이 직관적 (예: "버전 B가 A보다 나을 확률이 85%입니다.")

연속적 업데이트 가능 (데이터가 들어올 때마다 결과 갱신)

단점
사전 분포(prior)의 설정이 민감할 수 있음

✅ 3. Mann-Whitney U Test (Wilcoxon rank-sum test)
언제 사용하나요?

정규분포를 따르지 않는 두 그룹 비교

순위 기반 비교 (비모수 테스트)

t-test의 비모수 대안

✅ 4. Fisher’s Exact Test
소규모 샘플의 범주형 데이터 분석

chi-square test보다 정확

예시
클릭/미클릭 인원이 적은 경우

✅ 5. Bootstrapping (부트스트랩)
재표본 추출 기반의 신뢰구간 계산

장점

어떤 분포든 사용 가능

평균/중앙값/비율 등 다양한 통계량에 적용 가능

✅ 6. False Discovery Rate (FDR) 제어 (Benjamini-Hochberg 등)
동시에 여러 가설 검정할 때 유용

A/B/n 실험이나 여러 메트릭을 평가할 때

왜 필요한가요?

여러 번 테스트하면 우연히 유의미한 결과가 나올 확률 증가

✅ 7. Sequential Testing (연속 검정)
데이터를 계속 수집하면서 중간에 실험을 끝내도 되는 테스트

종류

SPRT (Sequential Probability Ratio Test)

Bayesian Sequential Test

A/B 테스트를 빠르게 끝내고 싶을 때 유용

✅ 8. Regression Analysis (회귀 분석)
A/B 효과를 조정변수와 함께 분석

사용자 지역, 디바이스, 시간대 등을 통제

예시

conversion ~ treatment + country + device

✅ 9. Uplift Modeling (Incremental Modeling)
**“누가 실험군에서 더 효과를 보는가?”**를 분석

A/B 테스트에서 사용자 세그먼트를 타겟팅할 때 유용
